## Rules

- Optimize all images for mobile delivery; WebP preferred for photos, SVG for icons and logos
- Use responsive `srcset` for large images
- Name files with kebab-case matching component or feature names
- Compress raster images; prefer SVGs for UI elements (small footprint, resolution-independent)
- Do not commit large uncompressed assets (>500 KB per file unless justified)
